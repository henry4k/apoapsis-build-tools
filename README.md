Apoapsis Build Tools
====================

Build tools used by several
[Apoapsis](https://github.com/henry4k/apoapsis)
related projects.


## Dependencies

- blender
- GIMP with the normalmap plugin
- [OpenImageIO](http://openimageio.org) and its Python bindings
